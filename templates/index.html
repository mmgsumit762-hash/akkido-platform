About Akkido Platform

Akkido Platform integrates ancient Vedic wisdom with modern financial analysis. The FinAstro Engine uses Panchang calculations (Tithi, Nakshatra, Yoga, Karana) to analyze historical stock price data and identify patterns based on lunar and solar positions.

Our backtesting engine downloads real historical OHLC data from Yahoo Finance and calculates Panchang values using Swiss Ephemeris for astronomical accuracy. Users can filter trading days by various Vedic factors to analyze performance patterns.

Key Features:

  • Real-time stock data via Yahoo Finance
  • Accurate Panchang calculations using Swiss Ephemeris
  • Customizable filtering by Vedic parameters
  • Comprehensive backtesting statistics
  • Responsive dark-mode UI
  • PythonAnywhere deployment ready
